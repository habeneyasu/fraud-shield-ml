# Models

This directory contains saved model artifacts from training.

## Contents

- **Trained Models**: Serialized model files (`.joblib`, `.pkl`)
  - Baseline models: `baseline_{dataset}_model.joblib`
  - Ensemble models: `ensemble_{type}_{dataset}_model.joblib`

## Usage

Models are saved automatically when using the model classes:

```python
from src.baseline_model import BaselineModel

baseline = BaselineModel()
# ... train model ...
baseline.save_model('models/baseline_ecommerce_model.joblib')
```

Models can be loaded later:

```python
from src.baseline_model import BaselineModel

baseline = BaselineModel()
baseline.load_model('models/baseline_ecommerce_model.joblib')
```

## Note

This directory is gitignored to avoid committing large model files. Models can be regenerated by retraining. For production deployments, use a model registry or artifact storage system.
