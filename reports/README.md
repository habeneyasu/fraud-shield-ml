# Reports

This directory contains generated model comparison reports and analysis outputs.

## Contents

- **Model Comparison Reports**: Detailed text and JSON reports from comprehensive model comparison scripts
  - Format: `model_comparison_{dataset}_{timestamp}.txt`
  - Format: `model_comparison_{dataset}_{timestamp}.json`

## Usage

Reports are automatically generated when running:

```bash
python scripts/comprehensive_model_comparison.py --dataset ecommerce
```

Reports include:
- Side-by-side model comparison tables
- Cross-validation results (mean Â± std)
- Model selection justification
- Recommendations and next steps

## Note

This directory is gitignored (except README.md) to avoid committing large report files. Reports can be regenerated by running the comparison scripts.
